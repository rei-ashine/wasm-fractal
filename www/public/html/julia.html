Julia Set

A complex number \( z_0 \)
is in the filled-in Julia set if,
as \( i \) → \( ∞ \), \( z_i \) does not
diverge where :

\[ \begin{cases} \begin{align} \space z_i &= z_{i-1}^2 + \space c \\ \\[0.01em] c \space &= -0.7269 + 0.1889 \space i \end{align} \end{cases} \]
